Project 2

Project 2 (3.5 points)

http://www.nand2tetris.org/02.php
This year you will also need to implement the following chips (as well as the ones given in the link above):
ChipDescription
Test script
Compare File
ShiftLeft.hdlThe chip will multiply by 2 its input.(The sign should not change)
ShiftLeft.tstShiftLeft.cmp
ShiftRight.hdlThe chip will divide by 2 its input.
The sign should not change.
Odd number are handle as follows:
-5/2 -> -3, 5/2-> 2
ShiftRight.tstShiftRight.cmp
Mul.hdlThe chip will multiply 2 numbers.
Mul.tstMul.cmp
Test file can be found here: test2.sh

Some notes:
  • Please see notes of project 1.
  • You can use any built in chip that you were asked to implemented in project 1 (and Xor).
  • You do not need to handle over flow in the mul, shiftLeft, and shiftRight chips, meaning that any number larger than 16 bits can be truncated to include only the 16 least significant bits.
